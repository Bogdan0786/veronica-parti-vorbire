Aragaze Aragazuri Este un substantiv neutru care face pluralul în „-e”, nu în „-uri”. Bareme Baremuri Deși sună similar cu alte cuvinte ce primesc „-uri”, pluralul corect este „bareme”. Boschete Boscheți Forma de plural masculin („boscheți”) este greșită; cuvântul este neutru. Chibrituri Chibrite Aici lucrurile stau invers: deși mulți spun „chibrite”, pluralul corect impune „-uri”. Chitare Chitări Pluralul corect acceptat este „chitare”, fără a schimba „a”-ul în „ă”. Colonei Coloneli Substantivul se termină în „-el”, iar pluralul regulat face „-ei” (ca la „băieței”). Monede Monezi Nu are loc o transformare din „d” în „z” la plural. Rămâne „monede”. Pancarte Pancărți Deși rimează cu „cărți”, singularul este pancartă, iar pluralul păstrează rădăcina: pancarte. Sindroame Sindromuri Corect gramatical este să formăm pluralul în „-oame”, asemenea altor cuvinte derivate. Sloganuri Slogane Adesea auzit la știri ca „slogane”, varianta acceptată de dicționar este cu „-uri”. Snoabe Snobe Adjectivul „snoabă” formează pluralul cu păstrarea diftongului „oa”. Succese Succesuri Probabil cea mai faimoasă greșeală. Cuvântul corect a fost și rămâne „succese”. Morminte Mormânturi Se formează prin alternanța „â” -> „i” și primește terminația „-e”. Bairamuri Bairame Acest cuvânt împrumutat din turcă formează pluralul în limba română cu „-uri”. Parbrize Parbrizuri La fel ca „aragaz”, acest substantiv neutru face pluralul în „-e”. Niveluri (stadii) Nivele Atenție! „Nivele” este pluralul pentru instrumentul de măsurat. Pentru grade, etape, stadii se folosește „niveluri”. Refrene Refrenuri Pentru versurile repetate ale unui cântec, folosim forma „refrene”. Feșe Fașe Forma corectă implică modificarea vocalei „a” în „e”. Vopsele Vopseluri Deși „vopseluri” este des folosit în popor, forma corectă literară este „vopsele”. Marmuri Marmure Pluralul care desemnează statuile de marmură sau varietățile materialului este „marmuri”.
